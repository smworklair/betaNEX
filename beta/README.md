# beta/ — учебная песочница: своя модель + ассистент поверх неё

Два связанных учебных подпроекта, отдельных от основного Go-бэкенда
NEX и от Python-сервиса `ai-gateway` (все три живут в `C:\betaNEX` как
соседи, ничего друг друга не ломает):

- **[`FrankAI/`](FrankAI)** — прототип собственной языковой модели:
  символьная RNN на чистом NumPy, обучаемая с нуля. Единый интерфейс
  `generate(prompt) -> text`.
- **[`Nex-pilot/`](Nex-pilot)** — ассистент, использующий `FrankAI` как
  движок по умолчанию (плюс промпт-шаблон, память диалога, CLI); может
  опционально работать через уже готовый `../ai-gateway` вместо FrankAI.

```
FrankAI  →  единый интерфейс Engine.generate()  →  Nex-pilot
                                                       │
                                     (опционально, вместо FrankAI)
                                                       ▼
                                                  ai-gateway → настоящий LLM
```

Начните с `FrankAI/README.md` (что за модель и её честные ограничения),
затем `Nex-pilot/README.md` (как это использовать как ассистента).
