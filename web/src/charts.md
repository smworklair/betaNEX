# web/src/charts.tsx

Набор лёгких графиков без внешних библиотек — всё нарисовано вручную на чистом SVG и стилизовано через CSS-переменные темы (`var(--accent)`, `var(--success)` и т.п.), поэтому графики автоматически подстраиваются под светлую/тёмную тему приложения. Существует, чтобы не тащить в проект тяжёлые charting-библиотеки ради простых визуализаций (кольца, столбцы, линии, спидометры, водопады), которые нужны в дашбордах финансов, учебного процесса и аналитики.

## Ключевое

- `Segment` — тип сегмента данных (`label`, `value`, `color`), общий для донат-диаграммы и легенды.
- `Donut` — кольцевая диаграмма с несколькими сегментами и опциональным текстом в центре.
- `Legend` — текстовая легенда к `Donut` (цветные точки + подписи, опционально значения).
- `Bars` — вертикальная столбчатая диаграмма.
- `Line` — одна линия тренда с заливкой площади под ней.
- `DualLine` — две линии на одном поле (например план vs факт), вторая — пунктиром.
- `Gauge` — кольцевой индикатор прогресса (спидометр) с процентом или произвольной подписью в центре.
- `HBars` — горизонтальные столбцы (рейтинги, структура затрат).
- `Waterfall` — водопадная диаграмма (bridge chart): как из начального значения через плюсы/минусы складывается итог.

## Как это работает

Каждый компонент — чистая функция от пропсов к SVG/HTML, без собственного состояния и без обращения к данным приложения — входные данные (`data`, `segments`, `steps`) передаются родителем. Геометрия строится вручную: например, `Donut` и `Gauge` используют `strokeDasharray`/`strokeDashoffset` на окружности для рисования дуг, `Line`/`DualLine` вычисляют полилинии по точкам, `Waterfall` считает накопительную сумму (`running`) по шагам и рисует каждый столбец от предыдущего накопленного уровня до нового, плюс пунктирные соединители между столбцами.

## Связи

Зависит только от React (`ReactNode`) и CSS-переменных темы, определённых где-то в глобальных стилях (`index.css`). Используется дашбордами и отчётными экранами по всему приложению — в частности ожидаемо в `web/src/pages/Dashboard.tsx`, `web/src/pages/accounting.tsx` и других местах, где нужна визуализация метрик.
