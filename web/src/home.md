# web/src/home.ts

Каталог «конструктора» главного экрана: какие крупные блоки (сводка дня, терминал, ярлыки разделов, список дел) и какие ярлыки быстрого перехода вообще существуют, чтобы пользователь мог включать/выключать и переставлять их местами. Как и `dock.ts`, это описание доступных вариантов, а не сама логика хранения выбора — выбор пользователя сохраняется в `prefs` (localStorage), как и для дока и верхней панели.

## Ключевое

- `HomeBlock`, `HOME_BLOCK_CATALOG` — каталог крупных блоков главного экрана (`brief` — сводка дня, `console` — встроенный терминал NEX, `shortcuts` — плитки ярлыков, `today` — список приоритетов дня), каждый с описанием.
- `DEFAULT_HOME_BLOCKS` — все блоки включены по умолчанию.
- `HOME_BLOCK_BY_ID` — тот же каталог блоков в виде словаря по id.
- `HomeShortcut`, `HOME_SHORTCUT_CATALOG` — каталог ярлыков быстрого перехода в разделы (финансы, студенты, журнал, задачи, почта, календарь, аналитика, безопасность, рассылка, приём, сообщество, уведомления) — каждый с иконкой и целевой страницей (`page`).
- `DEFAULT_HOME_SHORTCUTS` — 6 ярлыков по умолчанию.
- `HOME_SHORTCUT_BY_ID` — словарь ярлыков по id.
- `moveInArray(arr, item, dir)` — вспомогательная чистая функция: переставляет элемент массива на `dir` позиций (−1 — вверх/влево, +1 — вниз/вправо), используется для переупорядочивания блоков и ярлыков пользователем; возвращает новый массив, не мутирует исходный.

## Как это работает

Файл декларативный: содержит только каталоги и один универсальный helper для перестановки элементов в списке. Настоящая логика включения/выключения и сохранения порядка живёт в компоненте главного экрана (см. `web/src/pages/Home.tsx`) и в хранилище настроек — этот файл лишь поставляет список того, из чего можно выбирать, и значения по умолчанию.

## Связи

Зависит от иконок `lucide-react`. Напрямую используется в `web/src/pages/Home.tsx` — там строится сам главный экран и, вероятно, экран настройки его состава (см. также `web/src/pages/Settings.tsx`), опираясь на эти каталоги и на `moveInArray` для переупорядочивания.
