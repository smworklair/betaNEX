# web/src/pages/beta.tsx

Сборный файл с несколькими небольшими бета-разделами, которые пока не заслуживают отдельного файла: внутренний мессенджер, центр уведомлений, хранилище документов, студенческая лента, календарь-планировщик и облачное хранилище файлов. Все они работают на статичных моках (без реального бэкенда) и явно помечены бейджем `Beta`.

## Ключевое

- `Messenger()` — внутренний чат колледжа (список тредов + переписка), с кнопками «Составить ответ» / «Суммировать», которые дергают `llmAsk` из `../features/ai/llm` для генерации черновика ответа или сводки переписки. Адаптивен: на мобильном экране (`useIsMobile`) список и диалог показываются раздельно, с кнопкой «назад».
- `NotificationsPage()` — лента уведомлений с переключателем «Все / Непрочитанные», статичный список `NOTES`.
- `Documents()` — таблица документов с типами, размерами, авторами (включая «NEX (черновик)»).
- `Feed()` — студенческая лента объявлений (посты студсовета, кафедр, самого NEX).
- `Calendar()` — упрощённый недельный календарь с `AiBox` для планирования недели через ИИ.
- `Cloud()` — файловое хранилище организации (папки/файлы) с `AiBox` для поиска и суммаризации по файлам.
- `BetaNote({ text })` — локальный внутренний хелпер: карточка-плашка «Бета-функция» с пояснением, что данные демонстрационные.

## Как это работает

Все компоненты в этом файле — независимые страницы-разделы, объединённые общим паттерном: статичные моковые массивы данных (`THREADS`, `MSGS`, `NOTES`, `DOCS`, `POSTS`, `CAL_EVENTS`, `CLOUD_FILES`) наверху файла плюс минимальный локальный `useState` для интерактивности (выбор треда, вкладка уведомлений, лайки и т.п.). Реальных мутаций почти нет — большинство кнопок просто вызывают `toast('... — бета')` через `useApp()`, показывая, что функциональность ещё не реализована на бэкенде. Где уместно (мессенджер, календарь, облако), вкраплён живой вызов ИИ через `../features/ai/llm`, который либо даёт реальный ответ (если провайдер настроен), либо просит подключить ИИ в настройках.

## Связи

Зависит от `../ui` (`PageHead`, `Chip`, `Avatar`, `Beta`, `NexAsk`, `useApp`, `useIsMobile`), `./aibox` (`AiBox` — переиспользуемый ИИ-виджет) и `../features/ai/llm` (`llmReady`, `llmAsk`). Компоненты этого файла подключаются как отдельные маршруты в бета-навигации приложения; они не связаны общим стейтом друг с другом — каждый самодостаточен.
