# web/src/pages/CommandCenter.tsx

Экран «Командный центр» — главный аналитический дашборд системы, точка входа после логина для сотрудников (в отличие от `web/src/pages/Home.tsx`, который, судя по всему, ориентирован на более лёгкий/персональный обзор). Показывает разный набор виджетов в зависимости от роли: у администратора — полная административная панель с рекомендациями, графиками, аудитом и безопасностью; у остальных ролей (преподаватель, бухгалтер) — облегчённая версия под их задачи.

## Ключевое

- `CommandCenter` (default export) — точка входа: рендерит `AdminCenter` для роли `admin`, иначе `RoleCenter`.
- `AdminCenter` — полная панель администратора: сводка с прошлого входа, 4 KPI-плашки, блок «Рекомендованные действия» (`Rec`), три графика (`Charts`), блок «NEX действует сам» (лог автономных ИИ-действий), лента аудита, зона риска студентов (`AtRiskList` из `blocks.tsx`), неудачные входы, активные сессии, уведомления.
- `RoleCenter` — упрощённая панель для не-админских ролей: своя сводка, рекомендации (разные для бухгалтера и преподавателя через `isAcc`), 3 KPI.
- `Charts` — внутренний компонент трёх графиков: динамика посещаемости (`Line`), распределение оценок (`Donut`+`Legend`), студенты по группам (`Bars`) — данные считаются на лету из `groups`/`gradesFor`.
- `Rec` — переиспользуемая внутри файла карточка «рекомендованное действие»: иконка+заголовок+обоснование+кнопки действий, цвет по уровню серьёзности (`Severity`).
- `Kpi` — маленькая карточка метрики с трендом (стрелка вверх/вниз, дельта).

## Как это работает

Все данные на этой странице — моковые константы из `data.ts` (сессии, аудит, неудачные входы, уведомления) плюс несколько выведенных на лету агрегатов (распределение оценок через `gradesFor`). Действия кнопок в `AdminCenter`/`RoleCenter` в основном вызывают `toast(...)` (имитация выполнения) или `setPage(...)` (переход в раздел) — реальной мутации данных не происходит, это витрина сценариев «что должен уметь ИИ-ассистент организации». `AskHero` (из `ui.tsx`) — общий блок с полем быстрого вопроса к NEX, встроенный в оба варианта центра.

## Связи

Зависит от `web/src/ui.tsx` (`useApp`, `Chip`, `Avatar`, `Sparkline`, `NexAsk`, `AskHero`, `severityTone`, `severityLabel`), `web/src/blocks.tsx` (`AtRiskList`), `web/src/charts.tsx` (`Donut`, `Bars`, `Line`, `Legend`, `Segment`), `web/src/data.ts` (много моковых источников: `sessions`, `failedLogins`, `auditEvents`, `groups`, `gradesFor` и др.). Является одним из главных «показательных» экранов приложения — маршрутизируется из `App.tsx`.
