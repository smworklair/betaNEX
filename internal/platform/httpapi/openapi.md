# internal/platform/httpapi/openapi.go

Крошечный хендлер, раздающий встроенную в бинарь OpenAPI-спеку контракта API по HTTP.

## Ключевое

- `handleOpenAPI(spec []byte) http.Handler` — принимает готовые байты YAML-спеки и возвращает хендлер, который отдаёт их с `Content-Type: application/yaml; charset=utf-8`.

## Как это работает

Функция не парсит и не валидирует спеку — она просто отдаёт байты, переданные снаружи (обычно встроенные через `go:embed` в композиционном корне). Маршрут публичный и не требует аутентификации: спека не содержит пользовательских данных, а обратиться к ней должны в том числе внешние инструменты (генераторы клиентов, Bruno/Postman) ещё до входа в систему.

## Связи

Зависит только от `net/http`. Монтируется в routes.go как `GET /api/v1/openapi.yaml`, если `RouterConfig.OpenAPI != nil`; сами байты спеки поступают из композиционного корня `cmd/nexd/main.go` (обычно через `go:embed` файла спеки).

## На что обратить внимание

Маршрут сознательно не защищён авторизацией — это осознанное архитектурное решение, а не упущение: контракт API должен быть доступен инструментам интеграции до всякого логина.
